# 🖼️ Pasta de Imagens do Aplicativo

Esta pasta contém as imagens visuais do aplicativo Volei.

## 📋 Como Adicionar as Imagens

### 1. **Escudo Completo** (com texto VILA LINDA)
- **Nome do arquivo:** `escudo_vila_linda.png`
- **Tamanho recomendado:** 150-300px de largura
- **Onde aparece:**
  - Header (topo) - lado esquerdo
  - Header (topo) - lado direito
  - Seção de Perfil do Usuário (centralizado)

### 2. **Águia com Bola** (apenas a águia)
- **Nome do arquivo:** `aguia_volei.png`
- **Tamanho recomendado:** 280-300px de largura
- **Onde aparece:**
  - Seção de Lista de Convocação (esquerda)
  - Seção de Inscrição (direita)

## 📝 Instruções de Salvamento

1. **Clique com botão direito** em cada imagem que você recebeu
2. **Selecione "Salvar como imagem"**
3. **Navegue até:** `c:\Users\mateu\OneDrive\Documentos\GitHub\Volei\volei\assets\`
4. **Salve com o nome correto:**
   - Imagem 1 (escudo) → `escudo_vila_linda.png`
   - Imagem 2 (águia) → `aguia_volei.png`

## ✅ Verificação

Depois de salvar, você deverá ter:
```
volei/assets/
├── escudo_vila_linda.png
├── aguia_volei.png
└── README.md
```

## 🎨 Otimização de Imagens

Se as imagens ficarem muito pesadas, você pode:
- Usar https://tinypng.com para comprimir PNGs
- Redimensionar para 400px de largura máxima
- Formatos aceitos: PNG, JPG, JPEG, GIF, WebP

## 🔄 Reload do App

Depois de salvar as imagens:
1. **Atualize o navegador** (F5 ou Ctrl+R)
2. As imagens aparecerão automaticamente
3. Se não aparecerem, verifique os nomes dos arquivos

---

**Nota:** Se uma imagem não for encontrada, o app continua funcionando normalmente (não quebra o aplicativo).
